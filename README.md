# Connect 4 - Windows Forms (.NET)

## Description
This is a Connect 4 game developed using C# and Windows Forms (.NET). It is a two-player game where players take turns placing pieces into a 6x7 grid, aiming to connect four in a row horizontally, vertically, or diagonally.

The game uses a dynamically generated UI with buttons representing the board, and all game logic is handled through event-driven programming.

## Features
- 6x7 interactive game board created dynamically
- Two-player turn-based system (Red vs Yellow)
- Click-based column input system
- Automatic piece placement (gravity effect)
- Win detection:
  - Horizontal
  - Vertical
  - Diagonal (both directions)
- Automatic board reset after a win
- Simple Windows Forms UI
  
## Technologies Used
- C#
- .NET Framework (Windows Forms)
- Visual Studio

## Purpose
This project was created to practice:
- Object-Oriented Programming (OOP)
- Event-driven programming
- UI development with Windows Forms
- Game logic implementation in C#
